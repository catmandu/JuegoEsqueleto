JuegoEsqueleto
==============

Juego desarrollado en visual studio 2010, C# y XAML .NET 4 con kinect SDK

EL juego consiste en 15 preguntas definidas dentro del programa que apareceran aleatoriamente una vez comience el juego, cada una consta con 3 incisos de respuestas. Al acabar de responder todas, se informa el puntaje de preguntas correctas e incorrectas, para despues regresar al men√∫ principal.

Se puede usar un kinect para utilizar la mano del usuario como puntero, aunque se puede usar el puntero del mouse tambien si asi se desea.
